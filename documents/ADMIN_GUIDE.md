# Admin Guide

Welcome to the DarshanEase Admin Control Center. This guide will help you manage the platform.

## Accessing the Admin Panel
1. Log in using an account with the `ADMIN` role.
2. Click your profile icon in the Navbar and select **Admin Panel**.

## 1. Dashboard Overview
Provides a high-level summary of:
- Total Revenue
- Total Bookings
- Total Users
- Active Temples
- A 7-day trend chart of bookings
- Recent bookings list

## 2. Temple Management
- Add, Edit, or Delete temples.
- Update temple images, descriptions, timings, facilities, and deity information.

## 3. Slot Management
- Create daily darshan slots for specific temples.
- Define Slot Types (e.g., VIP, General), start/end times, and total capacity.
- **Note:** Bookings and cancellations automatically adjust the `availableSeats` for a slot.

## 4. Booking Management
- View all platform bookings.
- Search by Ticket ID, User, or Temple.
- Force-cancel bookings if necessary.
- **Export to PDF:** Click the export button to download a comprehensive report of all bookings.

## 5. Review Moderation
- Monitor all user reviews.
- Flag or delete inappropriate reviews.
- Deleting a review will automatically recalculate the temple's average rating.

## 6. User Management
- View all registered users.
- Elevate users to `ORGANIZER` or `ADMIN` roles.
- Delete malicious accounts.

## 7. Donation Management
- Track all donations across the platform.
- View Donor details, transaction IDs, and purposes.
- **Export to PDF:** Download detailed donation reports.

## 8. Settings
- Manage global platform configurations (General, Security, Notifications, System Data).
